# Portfolio Management Module

Este proyecto implementa un módulo de gestión de portafolio para una app de inversiones personales, cumpliendo con los requisitos de una fintech.

## Estructura del proyecto

- `main.py`: Punto de entrada para ejecutar el ejemplo principal.
- `src/entities/`: Contiene las entidades principales (`Stock`, `Portfolio`).
- `tests/`: Pruebas unitarias para validar la lógica de rebalanceo y cálculo de valor.
- `docs/`: Documentación, registro de la conversación, guía rápida y otros documentos relevantes.

## Documentación

- Consulta la documentación detallada en la carpeta [`docs/`](docs/):
  - [`full_conversation.md`](docs/copilot/FULL_CONVERSATION.md): Registro completo de la interacción y decisiones.
  - [`portfolio_design.md`](docs/copilot/PORTFOLIO_DESIGN.md): Diseño y justificación de la solución.
  - [`quick_start.md`](docs/QUICK_START.md): Guía rápida para ejecutar y testear el proyecto.

## Requisitos

- Python 3.8 o superior

## Ejecución rápida

Sigue los pasos de [`docs/quick_start.md`](docs/quick_start.md) para instalar dependencias, ejecutar el programa y correr los tests.
