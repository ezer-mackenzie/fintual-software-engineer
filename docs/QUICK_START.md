# Quick Start

Sigue estos pasos para ejecutar y probar el módulo de portafolio:

## Prerrequisitos

- Python 3.8 o superior instalado
- Git (opcional, para clonar el repositorio)

## Instalación

1. Clona el repositorio o descarga el código fuente.
2. Navega a la carpeta raíz del proyecto.

## Ejecución del programa principal

```bash
python main.py
```

Esto mostrará el valor total del portafolio y las acciones de rebalanceo sugeridas.

## Ejecución de los tests

```bash
python -m unittest tests/test_portfolio.py
```

Todos los tests deben pasar si la instalación es correcta.

## Estructura relevante

- `main.py`: Punto de entrada.
- `src/entities/`: Lógica principal.
- `tests/`: Pruebas unitarias.
- `docs/`: Documentación y registro de la conversación.
